Mobile Bug System Improvements project -- definition approved. 2-week project starting Monday with 1 DRI per platform. Adds a test-and-verify loop so the mobile auto-fixer can validate fixes against a simulated app -- today it operates on bug descriptions alone, which is why mobile has lagged.

DevEx Wins (cross-team)
✅
GHA pull_request workflow fix shipped across 9 repos. Tim Herby caught a long-standing CI gotcha where workflows were validating a synthetic merge preview rather than the actual branch -- 54 checkouts patched to use the immutable head SHA. Eliminates a class of phantom CI failures.
✅
mighty-mailgun skill shipped (Roque Santos). Bug bots can now query Mailgun directly for delivery, bounce, and suppression evidence -- closes the gap where Claude couldn't actually verify email-delivery hypotheses. Already ruled out Mailgun on BAN-12002 in minutes.

Android test coverage dashboard live. Mate's new coverage dashboard auto-recalculates on every merge. Coverage has climbed steadily since launch via the test-coverage campaigns tool.
